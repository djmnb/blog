# 护肤

## 护肤原理

护肤就像盖房子：先清理地基（清洁），再补充水分（补水），然后进行针对性加固（精华），最后盖上屋顶锁住营养（面霜），白天还要撑起一把伞（防晒）。

护肤步骤遵循一个基本原则：**质地从清爽到粘稠、分子从小到大。** 如果先把厚重的面霜涂在脸上，后续清爽的爽肤水或精华就很难被皮肤吸收了。

早晚护肤的核心区别在于：**白天重防护（阻挡紫外线），晚上重修护（深层清洁与滋养）**。

完整且科学的护肤流程：

**1.步骤一：清洁（洁面/卸妆）：**早/晚。

- **晚上：** 如果白天化妆或涂了防水型防晒，必须先用**卸妆产品**。接着使用**洁面乳**洗去残留的脏污和油脂。
- **早上：** 皮肤通常只分泌了少量油脂，用温和的氨基酸洗面奶、甚至直接用清水冲洗即可。

**2.步骤二：水乳调理（补水）：**早/晚。

洗完脸后角质层比较干燥，第一时间涂抹**爽肤水或柔肤水**。它可以快速为肌肤补充表层水分，并调整皮肤的酸碱度，让后续的营养成分更容易渗透。

**3.步骤三：密集修护（眼霜/精华）：**早/晚。

这是护肤中“功能性”最强的一步。

- **眼霜：** 眼周皮肤最薄，先涂眼霜可以避免后续面部精华误涂到眼周引起脂肪粒。
- **精华液：** 根据你的核心诉求（如美白、抗衰、控油、修护）选择对应的精华，局部或全脸涂抹。

**4.步骤四：锁水保温（乳液/面霜）：**早/晚。

前面补充的水分很容易蒸发，必须用含有油脂成分的**乳液或面霜**来“盖好屋顶”。油脂会在皮肤表面形成一层保护膜，把水分和精华的营养牢牢锁在里面。

**5.步骤五：日间防护（防晒）：**仅限白天。

**防晒是抗老最重要的一步！** 紫外线不仅会晒黑，还会破坏胶原蛋白导致皮肤松弛、长皱纹。白天出门前，务必在所有护肤步骤的最后涂抹**防晒霜**。

## 护肤产品名称

| **产品类别**      | **主要功效**                                                 | **适用时段与频次**                  |
| ----------------- | ------------------------------------------------------------ | ----------------------------------- |
| **卸妆水/油/膏**  | 溶解并卸除彩妆、防水防晒及皮肤深层油脂。                     | 仅在有彩妆/防水防晒的夜间使用       |
| **洁面乳**        | 洗去皮肤表面的汗液、灰尘和残留。                             | 每天早晚（干皮/敏感肌早上可用清水） |
| **爽肤水/精华水** | 快速浸润角质层，起到二次清洁或密集补水的作用。               | 每天早晚                            |
| **精华液**        | 高浓度活性成分，针对性解决抗老、美白或修护等问题。           | 每天早晚（部分A醇等成分仅限夜间）   |
| **眼霜**          | 滋润眼周，预防或改善眼部细纹、黑眼圈。                       | 每天早晚                            |
| **乳液 / 面霜**   | 建立皮脂膜，锁水保湿。偏油肌选清爽乳液，偏干肌选滋润面霜。   | 每天早晚                            |
| **防晒霜**        | 阻挡紫外线（UVA和UVB），防止皮肤晒伤和光老化。               | 仅限白天（出门前15-20分钟涂抹）     |
| **面膜 / 去角质** | 属于**特殊护理**。面膜提供即时密集补水；去角质帮助代谢老废角质。 | 每周                                |

> **初学者贴士：** 护肤千万不要盲目跟风“叠罗汉”。如果你是刚开始尝试，建议直接走最精简的路线：**洗面奶 + 保湿乳液/面霜 + 防晒霜**。只要这三步做好了，皮肤的基础状态就不会差。

## 肤质

### 1. 油性皮肤（油皮 / Oily Skin）

- **表现：** 皮脂腺分泌旺盛，全脸都容易出油，尤其是额头、鼻子和下巴（也就是常说的 **T区**）。
- **特征：** 容易长粉刺、黑头和痘痘，毛孔往往比较明显；但它有一个巨大的优势——因为油脂保护层厚，皮肤不易长皱纹，比较“耐老”。

### 2. 干性皮肤（干皮 / Dry Skin）

- **表现：** 皮脂腺“偷懒”，分泌的油脂太少，皮肤容易缺油缺水。洗完脸后如果不马上涂护肤品，会有明显的紧绷感，甚至脱皮。
- **特征：** 表面看起来毛孔细腻，几乎不长痘；但皮肤抵抗外界刺激的能力较弱，非常容易长干纹和细纹。

### 3. 中性皮肤（中性皮 / Normal Skin）

- **表现：** 皮肤界的“天选之子”。水油平衡达到了完美的境界，不干也不油。
- **特征：** 摸起来细腻有弹性，毛孔很小，极少长痘、脱皮或过敏，属于最健康的皮肤状态。

### 4. 混合性皮肤（混合皮 / Combination Skin）

- **表现：** 一张脸上同时存在“油”和“干”两种状态。通常表现为 **T区（额头、鼻翼、下巴）出油，但U区（两颊、眼周）偏干或正常**。
- **细分：**
  - **混油皮：** 整体偏油，T区大油田，两颊正常。
  - **混干皮：** 整体偏干，两颊干到紧绷，只有T区微微出油。

### 5. 敏感性皮肤（敏感肌 / Sensitive Skin）

- **表现：** 严格来说，敏感更像是一种**皮肤受损的状态**，但现在常被当作一种肤质。它的核心问题是“皮肤屏障”变得薄弱了。
- **特征：** 对外界刺激超级敏感。遇到换季、冷热交替、吹风或者用错护肤品时，脸上很容易出现发红、发烫、刺痛或发痒的反应。
